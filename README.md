## 📌 Project Overview

The Banking Analytics Dashboard is an end-to-end data analytics project designed to analyze customer accounts, banking transactions, loan portfolios, and branch performance. The project leverages SQL for data generation and analysis, Excel for reporting, Tableau and Power BI for interactive dashboards, providing actionable business insights to support decision-making.

---

## 🎯 Business Problem

Banks generate large volumes of transactional and customer data every day. This project helps answer key business questions such as:

- Which branches generate the highest revenue?
- Who are the top depositors?
- Which customers have the highest account balances?
- What is the monthly transaction trend?
- How are loans distributed across branches?
- Which customer segments are most profitable?

The project demonstrates relational database design using primary keys, foreign keys, and SQL joins.
